# GoodClean - 终端磁盘清理工具开发方案

## 一、可行性分析

### 技术可行性 ✅

* **Python + Textual**：Textual 是 Textualize 团队开发的现代 TUI 框架，基于 Rich 构建，提供丰富的交互组件（树形视图、表格、进度条、对话框等），非常适合构建此类工具

* **os.scandir() / pathlib**：Python 标准库提供高效的目录遍历能力

* **asyncio**：Textual 原生支持异步，可实现非阻塞扫描，UI 不卡顿

* **send2trash**：跨平台安全删除（移到回收站），防止误删

### 潜在风险与适配

| 风险               | 影响       | 适配方案                                      |
| ---------------- | -------- | ----------------------------------------- |
| Windows 系统目录权限不足 | 部分目录无法扫描 | try/except 捕获 PermissionError，标记为"无权限"并跳过 |
| 大目录扫描耗时长         | 用户体验差    | 异步扫描 + 实时进度条 + 增量更新结果                     |
| 符号链接导致无限循环       | 程序死循环    | 记录已访问路径，跳过重复目录                            |
| 误删重要文件           | 数据丢失     | 多重确认 + 默认使用回收站而非永久删除                      |
| 中文路径/文件名         | 显示乱码     | Python 3 原生 Unicode 支持，Textual 支持中文渲染     |

## 二、实现思路

### 核心功能模块

1. **目录扫描器** (`scanner.py`)

   * 递归扫描指定目录，收集文件/文件夹信息（路径、大小、类型、修改时间）

   * 异步执行，支持中断和取消

   * 处理权限错误和符号链接

2. **分析引擎** (`analyzer.py`)

   * 按目录层级聚合大小

   * 识别大文件（可配置阈值，默认 >100MB）

   * 识别垃圾文件模式（临时文件、缓存、日志、缩略图等）

   * 文件类型分布统计

3. **清理器** (`cleaner.py`)

   * 删除到回收站（默认）或永久删除

   * 清理前二次确认

   * 操作日志记录（方便撤销追踪）

4. **TUI 界面** (`app.py` + `widgets/`)

   * 主界面：目录树 + 大小排行 + 操作面板

   * 可视化：柱状图/表格展示大小分布

   * 交互：方向键导航，快捷键操作

### 快捷键设计

| 按键      | 功能              |
| ------- | --------------- |
| `↑↓`    | 导航目录树           |
| `Enter` | 展开/折叠目录         |
| `Space` | 选中/取消选中（用于批量清理） |
| `d`     | 删除选中项（移到回收站）    |
| `D`     | 永久删除（需二次确认）     |
| `s`     | 按大小排序切换         |
| `t`     | 按类型分类视图         |
| `r`     | 重新扫描            |
| `/`     | 搜索过滤            |
| `q`     | 退出              |
| `?`     | 帮助面板            |

### 垃圾文件识别规则

* **临时文件**: `*.tmp`, `*.temp`, `~*`, `*.bak`

* **系统缓存**: `Thumbs.db`, `desktop.ini`, `*.DS_Store`

* **开发缓存**: `node_modules/`, `__pycache__/`, `.pytest_cache/`, `target/`, `dist/`, `build/`

* **日志文件**: `*.log`

* **浏览器缓存**: Chrome/Firefox/Edge 缓存目录

* **系统临时目录**: `%TEMP%`, `%TMP%`

## 六、验证方式

1. **扫描验证**: 对已知大小的目录执行扫描，对比 `du` 或资源管理器显示的大小
2. **清理验证**: 选中文件删除后，确认回收站中有对应文件
3. **错误处理**: 扫描系统保护目录，确认不会崩溃且正确提示无权限
4. **性能测试**: 扫描包含 10 万+ 文件的目录，确认 UI 不卡顿且能显示进度
